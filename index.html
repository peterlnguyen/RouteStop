RouteStop

Plan your roadtrip!

Download this project as a .zip file Download this project as a tar.gz file

Step 1

Enter in your starting point, destination, and include in any stops along the way (e.g. "In-N-Out," "ATM," “gas station," or a valid address).

Step 2

RouteStop creates an A to B route using Google Maps, then boxes that route using Google’s RouteBoxer class, returning a list of northeast and southwest latitude and longitude coordinates that are used as the geographical search limits for the places of interest with CityGrid Places API. Please be patient, free hosting on Heroku takes a while!

Step 3

Route and waypoints are mapped with Google's optimized pathing.

Site is built with Node.js, Express, Stylus, Jade Template, sourced with GitHub, and hosted on Heroku.
